# mcp-servers.txt - MCP servers for Claude Code and Codex CLI
# Usage: install/claude.sh + install/codex.sh both read this file
#
# HTTP format:   <name> http <url> [auth=<source>] [extra args...]
# Stdio format:  <name> stdio cmd: <command...>
#
# Extra args on HTTP entries pass through to `claude mcp add`
# (e.g. --client-id <id>, --header "K: V"). Codex does not support these
# extras — install/codex.sh ignores them with a warning (servers using
# --client-id fall back to interactive OAuth on first Codex use).
#
# auth=<source> is a managed credential. Both Claude and Codex reconcile it
# on every sync. Sources:
#   auth=gh        — GitHub token via `gh auth token` (run `gh auth login` once).
#                    Claude: connection-time headersHelper (~/.claude/gh-mcp-headers.sh)
#                    Codex:  bearer_token_env_var = "GH_TOKEN", resolved by the
#                            codex() shell wrapper at launch — no token at rest
#   auth=context7  — CONTEXT7_API_KEY header from ~/.context7.env (optional:
#                    registers unauthenticated at lower rate limits when the env
#                    file is missing; create it with `bash install/auth.sh context7`)
# Auth-managed entries cannot also pass --header / --client-id extras.
#
# Install manually:
#   claude mcp add --transport <transport> --scope user <name> <url> [extra args...]
#   claude mcp add --scope user <name> -- <command...>

# Cloudflare: full API access (Pages, DNS, Workers, R2, Zero Trust, etc.)
# Write tools are gated by permissions.ask rules in home/dot_claude/settings.json.
cloudflare http https://mcp.cloudflare.com/mcp

# GitHub: PRs, issues, repo search, code review.
# OAuth/DCR is unsupported by GitHub's IdP, so we authenticate via gh CLI.
github http https://api.githubcopilot.com/mcp auth=gh

# OpenAI developer docs: API reference and guides for the OpenAI platform.
openaiDeveloperDocs http https://developers.openai.com/mcp

# Context7: up-to-date library documentation for any dependency.
# Remote streamable-HTTP endpoint (upstream-recommended over local npx stdio).
context7 http https://mcp.context7.com/mcp auth=context7

# DeepWiki: Devin-generated architecture wikis + Q&A for any public GitHub repo (no auth)
deepwiki http https://mcp.deepwiki.com/mcp

# Chrome DevTools: drive/inspect a live Chrome — network, console, performance
# traces. Google's official server; gracefully useless on machines without Chrome.
chrome-devtools stdio cmd: npx -y chrome-devtools-mcp@latest

# qmd: semantic+lexical search over the markdown knowledge layers (~/kb,
# Claude auto-memory, dotfiles docs). Talks to the warm local daemon
# (LaunchAgent dev.cade.qmd on macOS, lazy-start on Linux) — one model load
# shared by every agent session.
qmd http http://localhost:8181/mcp

# Blender: control Blender via MCP (requires addon.py installed in Blender — see github.com/ahujasid/blender-mcp)
blender stdio cmd: uvx blender-mcp
